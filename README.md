

fn-auth - v1.0.0
===
A very simple auth module.

Check demo/index.js for usage
## Install
### npm
```bash
$ npm install FireNeslo/fn-auth --save
```
## Usage
```js
const express = require('express')
const body = require('body-parser')
const auth = require('fn-auth')

const app = express()
  .use(body.json())
  .use(body.urlencoded())
  .use('/auth', auth())
  .use('/', express.static('public'))
  .listen(8080)

```
